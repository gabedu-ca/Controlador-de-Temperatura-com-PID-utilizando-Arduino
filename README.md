# Controlador-de-Temperatura-com-PID-utilizando-Arduino
Controlador de temperatura desenvolvido em Arduino utilizando algoritmo PID para manter um setpoint definido. O projeto simula uma estufa com modelo térmico, exibe dados em LCD e monitora temperatura, saída do controlador e setpoint em tempo real pelo Serial Plotter, demonstrando conceitos de controle de processos e automação industrial.
